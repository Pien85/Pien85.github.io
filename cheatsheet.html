CSS Selectors

Selector Meaning Example
Universal Selector Applies to all elements in the document *
Type Selector Matches elements names h1, h2, h3 etc
Class Selector Matches an element whose class = a value that matches the one specified after the period symbol .note, p.note etc
ID Selector Matches an element whose id attribute has a value that matches the one specified #intro

Other important CSS stuff I forget

Name Description
Selector Indicates which element a rule applies to. A rule can apply to more than one element: seperate them with comma's
Declaration Indicate how the element should bestyled. Declarations are split in to a property and a value these are separated by a colon (:)
Property Indicates the aspects of the element eg: color, font, width etc
Value Specify the settings suchs as 'yellow' or 'Times New Roman'
<link> Points an html file to the css stylesheet. Lives inside the <head> element and should have href, type and rel attributes
href Specifies the path to the css file
Type Specifies the type of document being linked. Value should be text/css
rel Specifies the relationship between the html page and the file it is linked to. Should be 'stylesheet' when linking to a css file
